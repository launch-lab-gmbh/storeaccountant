# StoreAccountant - WooCommerce plugin

![Header Image](docs/user/en/images/header.png)

> StoreAccountant is a WordPress/WooCommerce export plugin for accounting workflows. It
> ships with configurable accounting exports for orders, customers, and products.

## Features

### ✅ Order export

Export orders with configurable filters, columns, column sorting, and value mutators.

### ✅ Customer export

Export customers with configurable filters, columns, column sorting, and value mutators.

### ✅ Product export

Export customers with configurable filters, columns, column sorting, and value mutators.

### ✅ Export formats

Exports can be saved as:

- CSV
- JSON

### ✅ Password protected downloads

Export files are protected with password protection. You can share links with your accounting partners or other
third parties by providing the download link and the password.

### ✅ Extended permission assignment

You can assign additional permissions to backend roles for every action.
